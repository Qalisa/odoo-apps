## Note d'attention

Les procédures que vous trouverez dans cette documentation ont été réalisés sur un ordinateur Mac (équipé de MacOS) ainsi qu'à l'aide du navigateur Google Chrome.
Vous trouverez généralement des copies d'écran qui vous aideront à comprendre les étapes à suivre, et souvent au format `Smartphone`.

:::note
Si vous utilisez un écran d'ordinateur, il est possible que le rendu affiché sur votre écran soit légèrement différent.
:::
